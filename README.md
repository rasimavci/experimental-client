## Experimental Mobile WebRTC Client with Vuejs and vuex

> An easy example to build a client with Vue2

## Dependence

| Dependence | Version |
| ---------- | ------- |
| Vue        | 2.0+    |
| Vuex       | 2.0+    |
| Webpack    | 3.0+    |

## How To Use ?

```
step 1: yarn install

step 2:
    - develop: npm run dev
    - build  : npm run build, and use *lite-server* or *http-server* to visit the dist directory
```
